# -Django-
基于Django网页框架，开发一个对目标网页元素进行手动选择爬取的爬取器

python=3.8.5

Django=3.2.8

requests=2.24.0

beautifulsoup4=4.9.3

输入要处理的网址，获取目标网页，用鼠标勾选网页任意位置，获取该位置的HTML元素，如下图示例
